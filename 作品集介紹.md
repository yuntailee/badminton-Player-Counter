# 羽球比分計數器 - 作品集介紹

## 項目概述

這是一個專為羽毛球比賽設計的互動式比分計數與選手管理系統。採用純前端技術開發，提供直觀的視覺化球場介面，讓使用者能夠輕鬆管理選手、記錄比分，並追蹤比賽歷史與選手統計數據。

## 核心功能

### 1. 選手管理系統

- **新增選手**：快速添加參賽選手姓名
- **選手列表**：查看所有已註冊選手
- **拖拽定位**：直觀的拖放操作，將選手標籤拖至球場上方或下方區域
- **視覺化標記**：選手在球場內會自動變色標示（上方：橙色，下方：藍色）

### 2. 比分計數功能

- **即時計分**：支援 +1、-1 快速調整比分
- **手動編輯**：點擊分數可直接輸入數值
- **獨立重置**：可單獨重置任一方分數
- **全部重置**：一鍵清空所有分數

### 3. 比賽管理

- **比賽記錄**：自動保存每場比賽的詳細資訊（選手、比分、時間）
- **比賽結束**：達到21分後可結束比賽並自動記錄
- **場地交換**：一鍵交換雙方場地位置與分數
- **記錄刪除**：可刪除不需要的比賽記錄

### 4. 數據統計

- **選手統計**：點擊選手姓名查看個人數據
  - 勝場數、敗場數
  - 總場數、勝利率
  - 總得分、總失分
- **歷史追蹤**：側邊欄顯示所有比賽歷史記錄

## 技術亮點

### 前端技術

- **純原生開發**：使用 HTML5、CSS3、JavaScript（ES6+）實現，無需任何框架依賴
- **響應式設計**：完整支援桌面端、平板和手機，使用 Media Query 適配不同螢幕尺寸
- **互動體驗**：
  - 支援滑鼠和觸控操作的拖拽功能
  - 流暢的動畫過渡效果
  - 直觀的使用者介面設計

### UI/UX 設計

- **視覺化球場**：使用 CSS Grid 精確繪製標準羽毛球場地（包含邊界線、中線、發球線等）
- **漸層配色**：使用漸層背景提升視覺美感
- **互動反饋**：按鈕懸停效果、拖拽狀態提示、模態視窗設計

### 程式設計特色

- **狀態管理**：使用 JavaScript 物件管理選手、比分、比賽記錄等狀態
- **事件處理**：完善的滑鼠和觸控事件處理機制
- **數據計算**：自動計算選手勝率、總分等統計數據
- **記憶體管理**：高效的數據結構設計，支援快速查詢和更新

## 專案特色

1. **單文件應用**：所有功能整合在單一 HTML 文件中，易於部署和分享
2. **零依賴**：不依賴任何外部函式庫，載入速度快，相容性佳
3. **實用性強**：解決實際需求，適合羽球場館、俱樂部或個人使用
4. **擴展性**：程式碼結構清晰，易於後續功能擴展（如數據匯出、雲端同步等）

## 適用場景

- 羽球俱樂部比賽管理
- 個人練習記錄
- 業餘比賽計分
- 教學示範工具

## 技術棧

- **前端**：HTML5、CSS3、JavaScript (ES6+)
- **設計**：響應式設計、CSS Grid、Flexbox
- **互動**：原生 DOM API、事件處理、拖拽 API
